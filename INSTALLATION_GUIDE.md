## ⚠️ Avertissement normal d'Android

### Pourquoi Android affiche un avertissement ?

Lors de l'installation, Android affichera un message :

```
┌─────────────────────────────────────┐
│ ⚠️  Application de source inconnue  │
│                                     │
│ Cette application n'est pas sur le  │
│ Play Store. Installer quand même ?  │
└─────────────────────────────────────┘
```

**C'est NORMAL** car :
- ✅ L'app n'est pas sur le Google Play Store
- ✅ C'est une distribution directe (comme les apps d'entreprise, les bêtas, etc.)
- ✅ Toutes les apps hors Play Store affichent cet avertissement
- ✅ C'est une protection standard d'Android

### L'application est-elle sécurisée ?

**OUI !** Voici pourquoi :

|--------|--------|
| Code open-source | ✅ Disponible sur GitHub |
| Données personnelles | ✅ Aucune collecte |
| Permissions | ✅ Minimum requis (Internet uniquement) |
| Chiffrement | ✅ HTTPS forcé |
| Analyse antivirus | ✅ Aucun malware détecté |
| Signature numérique | ✅ APK signé avec clé release |

## 📲 Installation sur Android

### Étape 1 : Télécharger l'APK

1. Ouvrez votre navigateur Android
2. Accédez au site de téléchargement : `[VOTRE_URL]/download`
3. Le téléchargement démarrera automatiquement
4. Ou cliquez sur **"Télécharger APK"**

**Fichier téléchargé** : `edt_cnam_vX.X.X.apk` (environ 10-15 MB)

---

### Étape 2 : Autoriser les sources inconnues

Android bloque par défaut l'installation d'apps hors Play Store.

#### Sur Android 8+ (Oreo et plus récent)

1. Cliquez sur le fichier APK téléchargé
2. Android affichera : **"Pour des raisons de sécurité, votre téléphone n'est pas autorisé à installer..."**
3. Cliquez sur **"Paramètres"**
4. Activez **"Autoriser cette source"**
5. Retournez en arrière

#### Sur Android 7 et antérieur

1. Ouvrez **Paramètres**
2. Allez dans **Sécurité**
3. Activez **"Sources inconnues"**
4. Confirmez

**⚠️ Note** : Vous pourrez désactiver cette option après l'installation.

---

### Étape 3 : Installer l'application

1. Ouvrez le fichier APK téléchargé
   - Soit depuis les **notifications**
   - Soit depuis **Téléchargements** (app Fichiers)

2. Android affichera un écran de sécurité :
   ```
   ⚠️  Application de source inconnue
   
   Cette application n'est pas vérifiée par Google Play Protect.
   ```

3. Deux options possibles :

   **Option A** : Message simple
   - Cliquez sur **"Installer"**

   **Option B** : Message détaillé
   - Cliquez sur **"Plus d'infos"** ou **"Détails"**
   - Cliquez sur **"Installer quand même"**

4. L'installation démarre (quelques secondes)

5. Cliquez sur **"Ouvrir"** ou **"Terminé"**

---

### Étape 4 : Premier lancement

1. L'application se lance
2. Le calendrier se charge automatiquement
3. Vous pouvez maintenant consulter votre emploi du temps ! 🎉

## 🔄 Mises à jour

### Comment mettre à jour l'app ?

L'application vérifie automatiquement les mises à jour.

**Quand une mise à jour est disponible** :

1. Une notification s'affichera dans l'app :
   ```
   🎉 Mise à jour disponible
   
   Version actuelle : 1.1.32
   Nouvelle version : 1.1.33
   
   [Télécharger]  [Plus tard]
   ```

2. Cliquez sur **"Télécharger"**
3. La nouvelle version se télécharge
4. Installez-la par-dessus l'ancienne (vos données sont conservées)

**Vérifier manuellement** :
- Ouvrez le menu **⚙️ Paramètres**
- Cliquez sur **"Vérifier les mises à jour"**

## 🔧 Résolution de problèmes

### "Fichier APK introuvable"

**Solution** :
1. Vérifiez dans **Téléchargements** (app Fichiers)
2. Le fichier s'appelle `edt_cnam_vX.X.X.apk`
3. Retéléchargez si nécessaire

---

### "Installation bloquée"

**Solution** :
1. Assurez-vous d'avoir autorisé les **"Sources inconnues"**
2. Si le problème persiste, redémarrez votre téléphone
3. Réessayez l'installation

---

### "Application potentiellement dangereuse"

**C'est normal !** Voir la section [Avertissement normal d'Android](#-avertissement-normal-dandroid) ci-dessus.

**Si vous hésitez** :
- ✅ Consultez le code source sur GitHub
- ✅ Scannez l'APK avec VirusTotal.com
- ✅ L'app n'a accès qu'à Internet (aucune autre permission)

---

### "L'app ne se lance pas"

**Vérifications** :
1. Version Android minimum requise : **Android 5.0+** (Lollipop)
2. Espace disque disponible : **au moins 50 MB**
3. Connexion Internet : **requise pour le premier lancement**

**Solutions** :
1. Redémarrez votre téléphone
2. Désinstallez et réinstallez l'app
3. Vérifiez votre connexion Internet

---

### "Le calendrier ne charge pas"

**Vérifications** :
1. Vous avez une connexion Internet active
2. Les serveurs CNAM sont accessibles

**Solutions** :
1. Tirez vers le bas pour **actualiser**
2. Fermez et rouvrez l'app
3. Vérifiez votre connexion réseau (WiFi ou données mobiles)

## 🛡️ Sécurité et confidentialité

### Quelles données sont collectées ?

**AUCUNE donnée personnelle n'est collectée.**

| Donnée | Collectée ? |
|--------|-------------|
| Nom, prénom | ❌ Non |
| Email | ❌ Non |
| Localisation | ❌ Non |
| Contacts | ❌ Non |
| Photos | ❌ Non |
| Historique de navigation | ❌ Non |
| Données d'emploi du temps | ✅ Oui, mais stockées localement uniquement |

### Où sont stockées mes données ?

- **Emploi du temps** : Stocké localement sur votre téléphone
- **Préférences** : Stockées localement (dark mode, etc.)
- **Aucune donnée envoyée** : Rien n'est envoyé à des serveurs tiers

### Quelles permissions sont demandées ?

**Une seule permission** : `INTERNET`

**Pourquoi ?**
- Pour télécharger votre emploi du temps depuis les serveurs CNAM
- Pour vérifier les mises à jour de l'app

**Aucune autre permission** (pas de localisation, contacts, stockage, etc.)
